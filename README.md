# University Campus Network

## Overview
This project is a university campus network designed using Cisco Packet Tracer.

The network simulates a university environment with a main campus and branch campus, providing communication between different devices and departments.

## Features
- University campus network topology
- Main Campus and Branch Campus
- Connected PCs, servers, switches, and routers
- End-to-end connectivity between network devices
- Basic network security configuration
- Communication between different parts of the network

## Tools
- Cisco Packet Tracer

## Project Goal
The goal of this project is to design and simulate a functional university campus network that allows reliable communication between different network components.
